# stm32-sdk

Bare-metal C++20 SDK для микроконтроллеров STM32 с опциональной поддержкой
FreeRTOS. Только CMSIS, без HAL и LL.

## Что внутри

- **Драйверы**: GPIO, I2C, UART (interrupt и DMA TX), SPI, DMA, внутренний flash.
- **Сенсоры**: IMU MPU6050, OLED-дисплей SSD1306, SPI flash W25Q32.
- **Шаблоны проектов**: 7 готовых к сборке сценариев (bare-metal и FreeRTOS).
- **CLI `stmtool`**: создание проектов, сборка, прошивка, обновление SDK.
- **C++20-модули** во всех публичных API — без раздутых заголовков, детерминированный порядок компиляции.
